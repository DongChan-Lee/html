<[CSS 기초와 활용] 구글 클론 사이트 만들기! 개발자도구와 Bootstrap, codepen 활용>
: ! + Tab으로 VSCode 자동완성 html 가능
: <input> : 입력하는 칸을 만드는 태그
: <form> : 정보를 전송하는 태그
	--> action : 어디로 향할건지라는 주소
	--> method : 데이터의 전송 방식(GET : 주소창에 정보를 포함하여 전달하는 정보 전송 방식 / POST : 주소창에 정보를 표시하지 않고 내부적으로 정보를 전달하는 정보 전송 방식)

: Bootstrap : 이미 완성된 CSS들을 제공해주는 사이트
: Materialize : 이미 만들어놓은 버튼 제공해주는 사이트
: semantic-ui : 다 미리 만들어놓고 CSS만 가져다 쓰면 될 수 있도록
: codepen : 원하는 요소 검색해서 사용 가능(ex. css search bar)

: Shift + Tab : 들여쓰기 반대로
: placeholder : 입력 창에 써져있는 지워지는 글자(ex. Email을 입력하세요)


<[Javascript 기초와 활용 #1] 완전 쉬운 jQuery로 스타크래프트 만들기>
: <img> 이미지를 표시하는 태그 (src = 주소, alt = 이미지 설명)
: jQuery 사용방법(https://code.jquery.com/ 들어가서 최신버전 uncompressed 복사/붙여넣기)
: CSS는 head 태그에 link 태그를 통해 연결해주지만, JS는 body 태그가 끝나기 전에 CDN을 넣어줌

[Javascript를 사용하는 방법]
- 1. .js 파일을 만들고 html과 연결함
- 2. <script> 태그 사이에 바로 javascript 코드 작성

: A.ready(함수) : A가 준비(로딩)되었을 때 괄호 안쪽 함수가 실행되는 함수
: A.click(함수) : A를 클릭했을 때 괄호 안쪽 함수가 실행되는 함수
: toggle() : 나타난 것은 숨기고, 숨겨진 것은 나타나게 하는 것
: jQuery UI에서 이미 만들어져 있는 것들의 소스코드를 가져다 쓸 수 있음.
: codepen에서 javascript, jquery 소스코드도 사용할 수 있음.
